About This Project

This IoT Notice Board lets you send and display messages remotely using a web interface.
Features:

  • Secure login for authorized access
  • Send notices instantly to the board
  • View the last notices with timestamps
  • Powered by ThingSpeak IoT cloud
How it works: Enter your credentials, type your notice, and send! The message is stored in the cloud and can be displayed on a physical notice board using an IoT device.
The project is made by Suyash Ranshur, Pruthviraj, Muskan Sheikh ,Diksha .
